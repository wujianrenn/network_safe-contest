# 基于Vue3的后台管理系统+Vite2 

#### Project composition

vite2 + vue3 + vue-router4 + vuex4 + element-plus

#### Project setup

```
yarn install
```

#### Compiles and hot-reloads for development

```
yarn dev
```

#### Compiles and minifies for production

```
yarn build
```
